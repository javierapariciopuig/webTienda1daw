🔥 Firebase Configuration para Grand Slam Market

Pasos para configurar Firebase:

1. Crear proyecto en Firebase Console

Ve a Firebase Console y crea un nuevo proyecto llamado "grand-slam-market"

2. Añadir aplicación web

En tu proyecto Firebase, haz clic en "Añadir aplicación" y selecciona "Web"

3. Copiar configuración

Copia la configuración que te proporcione Firebase y pégala en el archivo firebase-config.js

4. Habilitar Firestore

En la consola de Firebase, ve a "Firestore Database" y crea una nueva base de datos en modo prueba

5. Probar conexión

Usa el botón de abajo para probar la conexión con Firebase
